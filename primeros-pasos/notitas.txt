
HTML: 
<href="#incio">  referencium a un elemento con id incio dentro de la pagina  -->> navegacion interna
<head> --> metadatos
<header> --> cabecera del sitio, con logo y barra de navegacion
<div class= "container"> --> contenedor que centra y limita el ancho de los elementos
<a class= "logo" href = "#inicio"> --> logo del sitio o titulo
<nav> + <ul><li><a>> -->barra de navegacion con links de enlace
<main> --> contenedor principal del sitio

<SECTION> AGRUPA ELEMENTOS DE UN TIPO   VS <ARTICLE> contenido independiente como una card con imagen y texto
<section class= "hero"> --> seccion inicial 
<section class ="section container" id="sobre" > --> seccion decriptiva sobre el proyecto
<section class= "cards"< --> seccion con tarjetas <articule class="card">
<section id= "links"> --> seccion con enlaces externos
<section id= "contacto"> --> seccion con formulario de contacto
<footer> --> pie de pagina
<TEXTAREA> "input mas grande" para formularios
<FORM> para formularios

crucial que cada <input> tenga un <label> asociado con el atributo for="id_del_input" para accesibilidad.
class para estilos repetidos (card, card-body).
id para identificadores únicos, útiles para links internos y CSS específico (id="inicio")

header, footer,nav, podria hacerlos aparte para reutilizarlos



CSS: 
en css si uso # para referenciar a un elemento con id y agrego estilos a ese elemento
